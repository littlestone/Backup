This sample shows how ASP.NET Web  Application will be created.



Building and Running the Sample Within Visual Studio

To build and run the Walkthrough_WebAppl_VB sample 

1. Open the solution (Walkthrough_WebAppl_VB.sln).

2 If there is a problem in opening the project, make sure you have defined "Walkthrough_WebAppl_VB" as Virtual Directory in
  IIS Web server. Use Internet Services Manager to define Virtual Directory.

3. In Solution Explorer, right-click the Walkthrough_WebAppl_VB project and click Properties. 

4. right-click the Reference and then Add Reference. Point the localtion of UODOTNET.DLL

5. Click OK. 

6. From the Debug menu, click Start Without Debugging.
