This sample shows how Windows Application will be created.



Building and Running the Sample Within Visual Studio

To build and run the Walkthrough_WindowsAppl sample 

1. Open the solution (Walkthrough_WindowsAppl.sln).
2. In Solution Explorer, right-click the Walkthrough_WindowsAppl project and click Properties. 

3. Right-click the Reference and then Add Reference... menu item. 

4. You will see Add Reference Dialog Box. Press "Browse..." Button. Locate UODOTNET.DLL assembly in your computer. 
   Generally it is found in C:\IBM\UNIDK\UODOTNET\BIN\UODOTNET.DLL
5. Click OK. 
6. From the Debug menu, click Start Without Debugging.
