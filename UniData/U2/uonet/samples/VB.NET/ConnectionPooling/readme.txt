This sample shows how ConnectionPooling Application will be created.



Building and Running the Sample Within Visual Studio

To build and run the ConnectionPooling sample 

1. Open the solution (ConnectionPooling.sln).
2. In Solution Explorer, right-click the ConnectionPooling project and click Properties. 

3. right-click the Reference and then Add Reference. Point the localtion of UODOTNET.DLL

4. Click OK. 
5. From the Debug menu, click Start Without Debugging.
